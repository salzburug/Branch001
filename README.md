## 外部ストレージへの配置
大容量ファイルはリポジトリに含めず、外部ストレージ（例: Google Drive, Dropbox, OneDrive）にアップロードしてリンクをREADMEに記載することを推奨します。

手順例（Google Drive, ブラウザ操作）:
1. ブラウザで Google Drive を開く。
2. `物体検知AI編/ObjectDetectionCourse.zip` をアップロードする。
3. アップロード後にファイルの共有リンクを取得する（リンクを「リンクを知っている全員に変更」などに設定）。
4. 下記の「外部ファイル」セクションに共有リンクを貼り付ける。
